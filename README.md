# lisbon_tourism_analysis
**End-to-end Python project exploring Lisbon’s tourism trends.** Led the technical execution: from data cleaning to visualization. Translated complex data into actionable marketing insights. Created as lead developer for NOVA IMS. #DataScience

## Project Goal
The objective of this analysis was to identify tourism patterns in Lisbon to support data-driven marketing strategies. We focused on tourist preferences and seasonal peaks to understand how to optimize visitor experiences.

## Technologies
* **Language:** Python
* **Libraries:** Pandas (Data Manipulation), Matplotlib & Seaborn (Data Visualization).
* **Environment:** Jupyter Notebook.

## Key Marketing Insights
* **Insight 1:** Identified key tourist segments based on behavior and preferences.
* **Insight 2:** Correlated seasonal trends with peak attraction visits.
* **Insight 3:** Proposed data-driven recommendations for local marketing campaigns.
